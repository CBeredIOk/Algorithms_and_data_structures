Задания из книги "Структуры данных и алгоритмы в JAVA" от Р. Лафоре

В данной книге есть множество заданий на определенные темы из глав книги. Примерами таких задач являются задача Иосифа Флавия, Ханойская башня, задача о рюкзаке, реализация циклического списка, стека, очереди и т.д.

В данном репозитории я загружаю свои решения данных задач, но задачи пока структурированы только по главам, поэтому поиск интересующей вас задачи в конкретной главе может быть затруднителен
